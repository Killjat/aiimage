📊 数据库表: image_gallery
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 核心字段
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  • id                - 主键
  • user_id           - 创建者ID
  • username          - 创建者用户名
  • model             - 图片生成模型 ✨
  • llm_model         - 使用的大模型 ✨
  • prompt            - 提示词
  • negative_prompt   - 反向提示词
  • image_url         - 图片数据 (Base64/URL)
  • image_size        - 图片尺寸
  • image_quality     - 图片质量
  • is_public         - 是否公开
  • views             - 浏览次数
  • likes             - 点赞数
  • description       - 图片描述
  • tags              - 标签
  • created_at        - 创建时间
  • updated_at        - 更新时间

🔍 索引
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✓ PRIMARY (id)
  ✓ idx_user_id (用户查询)
  ✓ idx_created_at (时间排序)
  ✓ idx_is_public (公开筛选)
  ✓ idx_views (热门排序)
  ✓ idx_likes (点赞排序)
  ✓ idx_model (模型筛选)
  ✓ idx_llm_model (大模型筛选) ✨
  ✓ ft_prompt (全文搜索)
  ✓ ft_tags (标签搜索)

🎮 API 端点
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  GET  /api/gallery/public              - 公开图片库
  GET  /api/gallery/user/:userId        - 用户图片库
  GET  /api/gallery/image/:imageId      - 图片详情
  GET  /api/gallery/search              - 搜索图片
  POST /api/gallery/image/:imageId/like - 点赞
  GET  /api/gallery/stats/models        - 模型统计
  GET  /api/gallery/stats/llm           - 大模型统计 ✨
