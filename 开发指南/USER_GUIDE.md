## 🚀 快速开始

### 首次使用

1. **打开应用**
   - 在浏览器中访问: http://localhost:5173
   - 确保后端服务正在运行

2. **选择功能**
   - 默认进入聊天界面
   - 点击 "🎨 生成图片" 进入图片生成界面

3. **开始使用**
   - 聊天：输入问题，按 Enter 发送
   - 图片：输入描述，点击生成

## 💬 聊天功能

### 基础使用

#### 1. 选择模型

在页面顶部的下拉框中选择 AI 模型：

- **🤖 Auto (自动选择最佳模型)** - 推荐新手使用
- **Grok 系列** - X.AI 的模型，擅长实时信息
- **Gemini 系列** - Google 的模型，多语言支持好
- **GPT 系列** - OpenAI 的模型，通用能力强
- **Claude 系列** - Anthropic 的模型，长文本处理好

💡 **提示**: 如果不确定选哪个，就用 "Auto"，系统会自动选择最合适的模型。

#### 2. 发送消息

1. 在底部输入框输入你的问题
2. 按 `Enter` 键发送（或点击发送按钮）
3. 等待 AI 回复

#### 3. 查看回复

- AI 的回复会实时显示在聊天区域
- 页面会自动滚动到最新消息
- 你可以继续提问，保持对话连续性

### 高级功能

#### 多轮对话

系统会自动记住对话历史，你可以：

```
你: 帮我写一个 Python 函数，计算斐波那契数列
AI: [提供代码]
你: 能优化一下性能吗？
AI: [提供优化后的代码]
你: 再加上单元测试
AI: [添加测试代码]
```

#### 对话历史

- 对话历史自动保存在浏览器中
- 刷新页面后历史仍然存在
- 清除浏览器数据会删除历史

#### 模型切换

- 可以随时切换模型
- 切换后对话历史保持不变
- 新消息会使用新选择的模型

## 🎨 图片生成

### 基础使用

#### 1. 打开图片生成界面

点击页面右上角的 "🎨 生成图片" 按钮

#### 2. 输入描述

在 "描述创意" 输入框中输入你想要的图片描述：

**简单描述**:
```
一只可爱的猫
```

**详细描述**:
```
一幅水彩画风格的插画，描绘一只橘色的猫咪坐在木质窗台上，
温暖的阳光从窗户照进来，在猫咪身上形成柔和的光影。
背景是模糊的室内场景，整体色调温暖明亮。
```

**专业描述**:
```
一个未来主义风格的城市景观，高耸的摩天大楼，飞行汽车在空中穿梭，
霓虹灯闪烁，赛博朋克风格，夜景，4K高清，电影级光影效果，超现实主义
```

#### 3. 选择模型

根据需求选择合适的模型：

- **Nano Banana 2** - 默认推荐，质量和速度都很好
- **Nano Banana** - 速度最快，适合快速预览
- **Flux 2 Pro** - 质量最高，适合专业作品
- **Flux 2 Flex** - 平衡选择，日常使用
- **Riverflow V2 Pro** - 文字渲染最好，适合海报
- **Riverflow V2 Fast** - 快速生成，适合批量

#### 4. 调整高级选项（可选）

点击 "⚙️ 高级选项" 展开：

**宽高比**:
- 1:1 - 正方形（Instagram、头像）
- 16:9 - 横屏（桌面壁纸、YouTube 封面）
- 9:16 - 竖屏（手机壁纸、Stories）
- 4:3 - 标准（传统照片）
- 3:4 - 竖版（海报、杂志）

**分辨率**:
- 1K - 快速预览、社交媒体（1024px）
- 2K - 标准输出、网页使用（2048px）推荐
- 4K - 高质量打印、专业作品（4096px）

#### 5. 生成图片

点击 "🎨 生成图片" 按钮，等待生成完成（通常 3-30 秒）

#### 6. 下载图片

生成完成后，点击 "📥 下载" 按钮保存图片

### 图片编辑

#### 上传图片

1. 点击 "📤 上传图片" 区域
2. 选择图片文件（或拖拽文件到区域）
3. 支持格式: PNG, JPG, WEBP, GIF
4. 最大文件大小: 5MB

#### 描述修改

上传图片后，输入框会变成 "✏️ 描述修改"：

```
例如：
- 将背景改为雪山
- 把天空变成日落
- 添加一只小鸟
- 改成水彩画风格
```

#### 生成新图片

点击 "✨ 开始编辑"，AI 会基于原图和你的描述生成新图片

## ❓ 常见问题

### 聊天相关

**Q: 为什么 AI 的回答不准确？**

A: 可能的原因：
- 问题描述不够清晰
- 选择的模型不适合这个任务
- 需要提供更多上下文信息

解决方法：
- 重新描述问题，提供更多细节
- 尝试切换其他模型
- 分步骤提问，逐步深入

**Q: 如何让 AI 记住之前的对话？**

A: 系统会自动记住当前会话的所有对话，你可以直接引用之前的内容。

**Q: 对话历史会保存多久？**

A: 对话历史保存在浏览器的 localStorage 中，除非你清除浏览器数据，否则会一直保存。

**Q: 可以导出对话记录吗？**

A: 目前版本暂不支持，V2 版本会添加此功能。

### 图片生成相关

**Q: 为什么图片生成失败？**

A: 可能的原因：
- 描述包含敏感内容
- 模型在你的地区不可用（403 错误）
- 网络连接问题
- 服务器繁忙

解决方法：
- 修改描述，避免敏感词
- 尝试其他模型
- 检查网络连接
- 稍后重试

**Q: 生成的图片不符合预期怎么办？**

A: 建议：
- 增加描述的详细程度
- 明确指定风格和细节
- 尝试不同的模型
- 多次生成，选择最好的

**Q: 如何生成高质量图片？**

A: 技巧：
- 使用 Flux 2 Pro 或 Riverflow V2 Pro
- 选择 4K 分辨率
- 提供详细的描述
- 指定专业的艺术风格
- 说明光影和构图

**Q: 可以生成包含文字的图片吗？**

A: 可以，但建议：
- 使用 Riverflow V2 Pro（文字渲染最好）
- 在描述中明确说明文字内容
- 文字不要太多（1-5个单词最佳）
- 使用简单的字体

**Q: 上传的图片会被保存吗？**

A: 不会。上传的图片只在生成时使用，不会保存到服务器。

## 🔧 故障排除

### 页面无法加载

**症状**: 打开页面显示空白或错误

**检查**:
1. 确认后端服务正在运行
   ```bash
   cd backend
   php -S 0.0.0.0:8080 -t public
   ```

2. 确认前端服务正在运行
   ```bash
   cd frontend
   npm run dev
   ```

3. 检查浏览器控制台是否有错误

### 无法发送消息

**症状**: 点击发送按钮没有反应

**检查**:
1. 确认输入框不为空
2. 检查网络连接
3. 查看浏览器控制台错误
4. 确认后端 API 可访问

### 图片生成失败

**症状**: 显示 403 或其他错误

**解决方法**:
1. 尝试其他模型
2. 检查 VPN 连接
3. 修改描述内容
4. 查看后端日志

### 对话历史丢失

**症状**: 刷新后对话消失

**原因**: 浏览器 localStorage 被清除

**预防**:
- 不要使用无痕模式
- 不要清除浏览器数据
- 定期导出重要对话（V2 功能）

### 性能问题

**症状**: 页面卡顿或响应慢

**优化**:
1. 清除浏览器缓存
2. 关闭其他标签页
3. 重启浏览器
4. 检查网络速度

## 💡 使用建议

### 聊天建议

1. **清晰表达**: 问题越清晰，回答越准确
2. **提供上下文**: 给出必要的背景信息
3. **分步提问**: 复杂问题分步骤解决
4. **验证答案**: AI 可能出错，重要信息需验证

### 图片生成建议

1. **详细描述**: 描述越详细，结果越好
2. **指定风格**: 明确艺术风格和质量要求
3. **合理预期**: AI 生成有随机性，多试几次
4. **保存作品**: 满意的图片及时下载保存

1. **使用 Auto**: 不确定时使用自动模型选择
2. **保存模板**: 常用提示词保存为模板
3. **批量处理**: 相似任务一次性完成
4. **快捷键**: 使用 Enter 快速发送
