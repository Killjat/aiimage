## 使用方式

### 前端选择模型

1. 打开图片生成器
2. 在模型下拉菜单中选择阿里巴巴模型（标签为 "阿里"）
3. 输入提示词
4. 点击生成

### API 调用

```bash
curl -X POST http://localhost:3000/api/image/generate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "prompt": "一只可爱的猫",
    "model": "alibaba-wan2.6-t2i"
  }'
```

## 常见问题

### Q: 如何在前端看到阿里巴巴模型？

A: 打开图片生成器，在模型下拉菜单中会看到所有可用模型，阿里巴巴模型会标记为 "阿里"。

### Q: 阿里巴巴模型和 OpenRouter 模型有什么区别？

A: 
- **阿里巴巴**: 国内模型，文字渲染强，访问速度快
- **OpenRouter**: 国际模型，多样性强，质量稳定

### Q: 如何选择最合适的模型？

A: 根据你的需求选择：
- 需要文字渲染 → 千问图像系列
- 需要最高质量 → 万相 2.6
- 需要快速生成 → 千问图像 2.0

### Q: 模型生成失败怎么办？

A: 
1. 检查提示词长度（不超过 2100 字符）
2. 检查网络连接
3. 查看错误日志
4. 尝试其他模型
