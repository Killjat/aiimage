# OpenRouter Chat System - Frontend

React + TypeScript 前端界面。

## 快速开始

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

前端将在 `http://127.0.0.1:5173` 启动。

## 功能

- 选择 AI 模型（Grok、Gemini、GPT-4）
- 发送消息并接收回复
- 显示对话历史
- 实时显示后端服务状态

## 注意事项

- 确保后端服务已启动（`http://127.0.0.1:8080`）
- 确保后端 `.env` 文件中已配置 OpenRouter API Key
